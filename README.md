# ASnowflake
A library which uses to generate Global Unique Ids based on snowflake,An alternative of UUID.

### Usage

```groovy
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
}
```

```groovy
dependencies {
	        compile 'com.github.zjutkz:ASnowflake:1.0.0'
}
```
